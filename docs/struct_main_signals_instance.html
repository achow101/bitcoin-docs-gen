Detailed Description

The MainSignalsInstance manages a list of shared_ptr<CValidationInterface> callbacks.

A std::unordered_map is used to track what callbacks are currently registered, and a std::list is to used to store the callbacks that are currently registered as well as any callbacks that are just unregistered and about to be deleted when they are done executing.
