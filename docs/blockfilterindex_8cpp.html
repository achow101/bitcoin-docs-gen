constexpr size_t CF_HEADERS_CACHE_MAX_SZ {2000}
constexpr

Maximum size of the cfheaders cache We have a limit to prevent a bug in filling this cache potentially turning into an OOM.

At 2000 entries, this cache is big enough for a 2,000,000 length block chain, which we should be enough until ~2047.
