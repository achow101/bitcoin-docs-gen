241  // Denial-of-service detection/prevention
242  // The idea is to detect peers that are behaving
243  // badly and disconnect/ban them, but do it in a
244  // one-coding-mistake-won't-shatter-the-entire-network
245  // way.
246  // IMPORTANT: There should be nothing I can give a
247  // node that it will forward on that will make that
248  // node's peers drop it. If there is, an attacker
249  // can isolate a node and/or try to split the network.
250  // Dropping a node for sending stuff that is invalid
251  // now but might be valid in a later version is also
252  // dangerous, because it can cause a network split
253  // between nodes running old code and nodes running
254  // new code.
